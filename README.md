# Landscape
Landscape Encryptor è uno script fatto da Itelcan3 & ! Giudizio in Python e Golang per aumentare la privacy e la sicurezza online rimanendo anonimi con vari strati di spoofing

# Metodi di spoofing
Ci sono diversi metodi che abbiamo attuato per fare in modo che non si riesca a penetrare e riottenere la stringa originale dopo l'hashing...
Alcuni di questi metodi sono: spoofing del master fingerprint, generazione di credenziali estese e generazione di seed passati da 20 sottostrati...

# Come usare
Per adesso ti consigliamo di usare il file ```Landscape.py```. Ciò che devi fare è soltanto installare la libreria di faker con ```pip install faker```. La versione in Golang è ancora molto sperimentale e soprattutto sprovvista di funzione per la generazione di credenziali fake (email + password). Per tanto, ti consigliamo di usare i file .exe pre-compilati o di usare gli script disponibili nel codice

# Aggiornamenti
Ovviamente, con il progresso della cyber security in futuro farò molti aggiornamenti riguardo a questo codice già impacchettato. Una delle funzioni che vorrò infatti aggiungere nel futuro è lo spoofing dell'attività Discord (RPC). Perciò rimanete aggiornati!
